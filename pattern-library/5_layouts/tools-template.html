Add New Post Type

Template Structure Settings Stages

A post type is a custom post structure that is created by a deployment administrator. To add a new post type, you can choose from the below options or start from scratch by selecting Custom Structure:

Basic Post

  • Title
  • Description
  • Location
  • Link(s)
  • Photo(s)

Photo Check-in

  • Title
  • Description
  • Photo(s)

Missing Person

  • Title
  • Name of missing person
  • Description
  • Location
  • Link(s)

Unstable Building

  • Title
  • Description
  • Location
  • Link(s)

Useful Links

  • Title
  • Description
  • Link(s)

Custom Structure

Please choose select the structure of your post type before saving.
